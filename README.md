Proiect: Aplicație de gestiune a unei companii aeriene
Tehnologii: C#, .NET Framework, Windows Forms, Visual Studio 2022

Descriere:
Aplicație dezvoltată pentru gestionarea operațiunilor unei companii aeriene, incluzând: zboruri, pasageri, bilete și anulări. Fiecare formular și funcționalitate este concepută pentru a oferi o experiență de utilizare intuitivă și eficientă, permițând administrarea rapidă a informațiilor și proceselor specifice.

Responsabilități / realizări:
-Dezvoltarea interfeței grafice cu Windows Forms
-Implementarea funcționalităților CRUD pentru zboruri, pasageri și bilete
-Gestionarea evenimentelor și validarea datelor în aplicație
-Testarea aplicației pentru asigurarea funcționării corecte a tuturor operațiunilor
